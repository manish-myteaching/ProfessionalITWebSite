HTML Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It describes the structure of a Web page semantically and originally included cues for the appearance of the document.

Basic Structure:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Web Page</title>
</head>
<body>
  <h1>Welcome to HTML!</h1>
  <p>This is a paragraph of text.</p>
</body>
</html>

Every HTML document starts with a <!DOCTYPE html> declaration, followed by the <html> root element. The <head> contains meta-information, and the <body> contains the visible page content.
